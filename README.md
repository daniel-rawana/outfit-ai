## ğŸš€ Getting Started

1. Clone the repo  
```bash
git clone https://github.com/daniel-rawana/outfit-ai/
cd outfit-ai
```

2. Install backend dependencies  
```bash
cd backend
pip install -r 'requirements.txt'
```

2. Install frontend dependencies  
```bash
# from outfit-ai directory
cd frontend
npm install
```

3. Run the backend  
```bash
flask run
```

3. Run the frontend  
```bash
npm start
```

4. Navigate to `http://localhost:3000`
